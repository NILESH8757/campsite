<h2>Campsite </h2>
<p>
A website that rates and reviews nature camping sites. For now, UI is kept simple using HTML/CSS, Bootstrap and backend is developed using NodeJs, ExpressJs with PassportJs for Authentication. No-SQL database (MongoDB) is used. 
</p>
